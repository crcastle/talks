Web Application Framework

A software framework, designed to support the development of web applications by alleviating the overhead associated with common activities performed in web development.
  • Architecture - To encourage best-practice, MVC for example.
  • Common idioms - Don't waste time making trivial choices.
  • Templates - Some even include data-binding.
  • Caching - Because fetching the same thing twice is silly.
  • Routing - /take/me/somewhere/cool
